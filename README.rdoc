=== Simple Task Management App

This Rails project is a simple task management app built using <tt>Rails 4.2.6</tt> & <tt>SQLite3</tt>:

* There are User
* Each User could have many Tasks
* User can Create/Update/Delete the Tasks

== How to Setup

1. `bundle install`
2. `rake db:migrate`
3. `rails s`

After that you could open the app on <tt>http://localhost:3000</tt>
